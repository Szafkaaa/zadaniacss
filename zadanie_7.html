Zadanie 7

  1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego?

    a) Styl lokalny pozwala na formatowanie konkretnego elementu na stronie internetowej, nie wpływając na inne elementy.

  2. Składnia teoretyczna:

                
<style type="text/css">
.localStyle {
    definition: value;
}
</style>

  3. Co może być selektorem?

    b) Selektorami mogą być nazwy klas, identyfikatory, tagi HTML lub kombinacje tych elementów.

  4. Co wipisujemy jako "cechę". Jakie jest znaczenie średników?

    c) Jako "cechę" wpisujemy właściwość CSS, czyli to, co chcemy zmienić w wybranym elemencie. Średniki oddzielają deklaracje stylu.

  5. Co wipisujemy jako "wartość" (w deklaracji stylu)?

    b) Jako "wartość" w deklaracji stylu wpisujemy wartość, jaką chcemy przypisać wybranej właściwości CSS dla danego elementu.
